4. AUTO-FLOAT MECHANISM
--------------------------------------------------------------------
Plan: on entering manual mode, run
  hyprctl keyword windowrulev2 "float,class:.*"
and float every currently-open tiled window via a loop over
`hyprctl clients -j`. On exit, unset the rule and leave already-
floating windows as they are (no forced re-tile).

  Before building this, check whether toggle_workspace_float.sh or
  toggle_global_float.sh (hypr/) already do something close enough
  to reuse instead of writing a new mechanism.
  Checked?  [x] yes, reuse: ________________   [ ] no, write new

  On exiting manual mode, should windows that were floated
  automatically snap back to tiled?
  [x] yes, re-tile everything (order not guaranteed to match before) EXCEPT FOR THE SPECIAL WORKSPACE. THOSE SHOULD STAY FLOATING
  [ ] no, leave them floating until manually tiled (default)
